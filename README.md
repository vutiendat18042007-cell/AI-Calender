# AI-Calender
A local project  
Aim: Create a AI powered by other API key to assist creating events that people are too lazy to create manually or just want to make things easier to arrange their plans.  
Detail: Use whisper (from speech to text) to transfer from the human's voice to a effortless command, which AI can just pull that to calendar for you, so that anyone can use it without worrying about wasting time to do by fingers.  
How to use: You need to create your own API key from any platform(OpenAI, Gemini,...) then copy-paste your API key to API input. Then speak to create text or type your own command to run the AI, the result will appear below. Click the "Add to Google Calendar" button then it will appear in your Calendar account.
